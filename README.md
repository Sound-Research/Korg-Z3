# Korg Z3

Documentation for the Korg Z3 guitar synthesizer.

## Specifications

- **Type**: Guitar synthesizer, divided-pickup driven
- **Manufacturer**: Korg
- **Released**: 1989
- **Sound source**: Yamaha YM2414 (OPZ) 4-operator FM, 8-voice polyphonic
- **Voices**: 128 Programs + 128 internal Sounds, 8 banks × 8 patches
- **Pickup driver**: ZD3 (24-pin connector)
